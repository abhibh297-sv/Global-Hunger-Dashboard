
# Global Hunger Index - Excel Data Analytics Project

## 📌 Project Overview
This project is based on the **Global Hunger Index dataset** and was created using **Microsoft Excel**.  
The aim of this project is to analyze hunger levels across different countries and regions using data cleaning, visualization, and dashboard creation techniques.

## 🎯 Objectives
- Understand global hunger trends
- Compare countries based on hunger scores
- Identify top affected regions
- Build an interactive dashboard for insights

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Conditional Formatting
- Data Cleaning Techniques

## 📊 Features
- Cleaned and organized raw dataset
- Dynamic Pivot Tables
- Charts for comparison and trends
- KPI cards for quick insights
- Interactive Dashboard

## 📈 Key Insights
- Some regions have significantly higher hunger scores
- Major differences exist between countries
- Visual data helps in better understanding global issues
